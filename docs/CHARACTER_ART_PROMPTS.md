# MathQuest Character Art Prompts

## Art Specifications (All Characters)
- **Format**: PNG with transparent background
- **Size**: 1024×1024 pixels
- **Style**: Bright, colorful cartoon illustration suitable for ages 5-11
- **Centering**: Character centered in frame
- **NO** text, labels, or background scenery

---
