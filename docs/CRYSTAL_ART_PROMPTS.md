# MathQuest — Crystal Art Prompts

## Art Specifications (All Crystals)
- **Format**: PNG with transparent background
- **Size**: 1024×1024 pixels
- **Style**: Bright, colorful cartoon illustration matching the MathQuest fantasy game style (ages 5–11)
- **Centering**: Crystal centered in frame, floating with a faint magical glow
- **NO** text, labels, or background scenery
- **Each crystal** should feel magical, ancient, and precious — like a gem that holds immense power
- **Math symbol** carved/glowing on the front face of each crystal

**Save as**: `/assets/crystals/{id}.png` (e.g. `crystal-addition.png`)

## MINIATURE FLOATING CRYSTAL VARIANTS

For the avatar display, each crystal also needs a smaller, simpler variant (used at ~32–48px) that reads well at tiny sizes. These can be generated from the same prompts above but with this additional instruction appended:

```
MINIATURE VARIANT: Make the crystal simpler with fewer facets and bolder colors.
Emphasize the glow and silhouette over fine detail. The symbol should be clearly
visible even at very small sizes (32px). Slightly more rounded/soft edges.
Add a gentle bobbing animation feel — crystal tilted slightly as if floating.
Save at 256x256 for the mini variant.
```

**Save mini variants as**: `/assets/crystals/{id}-mini.png`
